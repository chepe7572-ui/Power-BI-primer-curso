# 📊 Curso Básico de Power BI

Bienvenido/a a este repositorio del **Primer Curso de Power BI**.  
Este proyecto está diseñado para introducir los conceptos fundamentales de Power BI, desde la carga de datos hasta la creación de dashboards interactivos.

---

## 🎯 Objetivo del Curso

El objetivo principal es que el estudiante:

- Comprenda qué es Power BI y para qué se utiliza.
- Aprenda a importar y transformar datos.
- Cree modelos de datos básicos.
- Diseñe visualizaciones claras y profesionales.
- Construya dashboards interactivos.
- Publique y comparta reportes.

---

## 🧠 ¿Qué es Power BI?

Power BI es una herramienta de inteligencia de negocios (Business Intelligence) desarrollada por Microsoft que permite transformar datos en información visual e interactiva para apoyar la toma de decisiones.

---

## 📚 Contenido del Curso

### 🔹 Módulo 1: Introducción a Power BI
- Instalación de Power BI Desktop
- Interfaz y componentes principales
- Conceptos básicos de BI

### 🔹 Módulo 2: Importación de Datos
- Conexión a archivos Excel
- Conexión a bases de datos
- Conexión a archivos CSV

### 🔹 Módulo 3: Transformación de Datos (Power Query)
- Limpieza de datos
- Eliminación de valores nulos
- Creación de columnas personalizadas
- Tipos de datos

### 🔹 Módulo 4: Modelado de Datos
- Relaciones entre tablas
- Tipos de relaciones
- Introducción a DAX
- Medidas básicas

### 🔹 Módulo 5: Visualización de Datos
- Gráficos de barras
- Gráficos de líneas
- Tablas y matrices
- Segmentadores (Slicers)
- Diseño de dashboards

### 🔹 Módulo 6: Publicación y Compartición
- Publicación en Power BI Service
- Compartir reportes
- Actualización de datos
